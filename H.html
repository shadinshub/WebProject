Hyper Text Markup Language (HTML)

The foundation of every website in the world.

What is HTML?

HTML stands for HyperText Markup Language. It is used to design web pages using markup.

Why HTML?

Without HTML, web pages would not exist. HTML structures content for browsers.

HTML Versions

HTML has evolved over time, from HTML 1.0 to the latest HTML5.

Facts About HTML?

  • Purpose: Structures web content (text, images, links).
  • Uses Tags: Elements like h1, p, a define content.
  • Attributes: Extra info inside tags
  • HTML5: Latest version (released 2014) with new semantic tags.
  • Basic Structure: Requires !DOCTYPE html, html, head, body elements
  • Not a Programming Language: No logic/calculations (unlike JavaScript)
  • Works with CSS/JS: Styled with CSS, made interactive with JavaScript

HTML Key Features

Feature Description
Semantic Tags Improves SEO and code readability.
Forms Collects user input easily.
Multimedia Supports images, audio, and video.
